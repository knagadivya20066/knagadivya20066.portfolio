# ✨ Kalugotla NagaDivya — Portfolio

> *"Turning lines of code into experiences that feel a little like magic."*

A fully hand-coded personal portfolio with a **Digital Theme Park** concept — built with zero frameworks, zero templates. Just pure HTML5, CSS3, and Vanilla JavaScript.

🔗 **Live Site:** [knagadivya20066.github.io](https://knagadivya20066.github.io)

---

## 🎢 Sections

| Section | Theme Name | What's Inside |
|---|---|---|
| Hero | Welcome Gate | Animated intro, name, tagline |
| About | Storybook Section | My story, traits, hackathon background |
| Skills | Magic Powers Zone | Floating skill badges by category |
| Projects | Adventure Rides | AI Resource Optimizer, Scheme Eligibility Checker |
| Education | The Journey | Timeline of education & achievements |
| Contact | Exit Portal | Contact form + social links |

---

## ⚡ Features

- 🌌 **Animated starfield background** — 220 twinkling particles built on Canvas API
- 🖱️ **Custom cursor** — lavender dot with a smooth trailing ring
- 🎞️ **Scroll-triggered reveals** — elements fade in as you scroll using IntersectionObserver
- 🏷️ **Floating skill badges** — subtle CSS keyframe animations
- 📱 **Fully responsive** — works on mobile and desktop
- 🎨 **Dark cosmic theme** — deep navy + lavender + sky blue + gold palette
- ✉️ **Contact form** with animated submit button

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, keyframes, backdrop-filter)
- Vanilla JavaScript (no frameworks)
- Google Fonts — Cormorant Garamond, Outfit, Dancing Script
- Canvas API (starfield)
- IntersectionObserver API (scroll reveals)
